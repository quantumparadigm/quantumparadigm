Javascript

Once you've added or removed your sections you need to go to main.js located in js folder.And look for the //MULTISCROLL function. And add or remove anchors according to the numbers of sections you have. 

        
          
    //MULTISCROLL
    $('#main-container').multiscroll({
        sectionsColor: [ 'white','white','white','white','white','white'],
        anchors: [ 'first','second','third','fourth','fifth','sixth'],
        easing:'easeInOutCubic',
        menu:'.menu-left',
        scrollingSpeed: 1000
        ...
    
    });

For example we need 5 sections so we have 5 anchors 

        
          
    //MULTISCROLL
    $('#main-container').multiscroll({
        sectionsColor: [ 'white','white','white','white','white'],
        anchors: [ 'first','second','third','fourth','fifth'],
        easing:'easeInOutCubic',
        menu:'.menu-left',
        scrollingSpeed: 1000
        ...
    
    });

For a correct responsive function,add or remove this lines depending of the number of sections you have (in this example we have 5 sections).This functions are located at the bottom of the file multiscroll.js 

        
          
    $.fn.multiscroll.destroy = function() {

        $("#right1").insertAfter("#left1");
        $("#right2").insertAfter("#left2");
        $("#right3").insertAfter("#left3");
        $("#right4").insertAfter("#left4");
        $("#right5").insertAfter("#left5");

    $.fn.multiscroll.build = function() {

        $("#right1").prependTo("#right-part");
        $("#right2").prependTo("#right-part");
        $("#right3").prependTo("#right-part");
        $("#right4").prependTo("#right-part");
        $("#right5").prependTo("#right-part");

Add projects

Add a new link in projects-list list-a , we can name it projects-link-4.Next add a new section,in this case section-five . 

            

       <div class="projects-list list-a">
           <a class="projects-link-4"></a>
       </div>
       ...
       <div class="section-five p-h1 display-none">
            
       </div>

Now go to main.js and duplicate or add this code with the new names. 

            
    
    //FADE IN-OUT PROJECTS SECTIONS
    $('.projects-link-4').on("click", function(e) {
        if (nodoubleclick) return;
        nodoubleclick = true;
        e.preventDefault();
        $('.section-one').fadeOut( 650, function(){
            $('.section-five').addClass('active-section');
            $('.section-five').hide().fadeIn(850);
            if ($(window).width() > 1024) {
            $('#left4').vegas('options', 'transition', 'slideLeft2').vegas('jump',1);
            $('#right4').vegas('options', 'transition', 'slideRight2').vegas('jump',1);
            }
        });
    });

Remember : vegas must have the same number as your section projects,in this case #left4 and #right4

Contact form


Open up the file mailer.php located in js folder and replace youremail with your email to receive messages from the contact form. 

          

<?php
if(empty($_POST['name'])   ||   
    empty($_POST['email'])  ||
    empty($_POST['message'])||  
    !filter_var($_POST['email'],FILTER_VALIDATE_EMAIL))   
  {    
       echo "No arguments Provided!";   return false;   
  }
     
  $name = $_POST['name'];
  $email_address = $_POST['email'];
  $message = $_POST['message'];     

 $to = 'youremail';

 $email_subject = "Contact form submitted by:  $name"; $email_body = "You have received a new message. \n\n".                
                   " Here are the details:\n \nName: $name \n ".                 
                   "Email: $email_address\n Message \n $message";
 $headers = "From: youremail\n";
 $headers .= "Reply-To: $email_address";    
 
 mail($to,$email_subject,$email_body,$headers); return true;            
?>

Remember : Contact form doesn't work locally.
